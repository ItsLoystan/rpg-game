# rpg-game

## Goal

Defeat all six enemies by answering questions about AI correctly.

## Controls

WASD - Movement
E - Interact with NPC
Left Click - Close dialogue/Choose Action

## Recommendations

Interact with the glasses NPC first at spawn point
